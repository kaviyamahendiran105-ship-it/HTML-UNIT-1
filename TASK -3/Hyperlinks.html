HTML LISTS COMPLETE GUIDE

1. Ordered List

Definition: Shows items in numbered order.

Command:

<ol>
  <li>Item</li>
</ol>

Example:

  1. Login
  2. Select course
  3. Start learning

2. Ordered List Types

Command: <ol type="A">

  1. HTML
  2. CSS
  3. JS
  1. First
  2. Second

3. Unordered List

Definition: Shows items using bullets.

Command:

<ul>
  <li>Item</li>
</ul>

Example:

  • Apple
  • Mango
  • Banana

4. Bullet Styles (List Styling Basics)

Command: <ul type="square">

Disc:

  • Item
  • Item

Circle:

  • Item
  • Item

Square:

  • Item
  • Item

No Bullet:

  • Home
  • About

5. Nested List (List inside list)

Command:

<ul>
  <li>Item
     <ul>
        <li>Subitem</li>
     </ul>
  </li>
</ul>

Example:

  • Fruits
    • Apple
    • Mango
  • Vegetables
    • Carrot
    • Potato

6. Mixed Nested List (Ordered + Unordered)

  1. Frontend
    • HTML
    • CSS
  2. Backend
    • Python
    • Node

7. Extended List (Description List)

Definition: Used for terms and descriptions.

Commands:

<dl>  Definition List
<dt>  Term
<dd>  Description

Example:

HTML
Language used to create webpages
CSS
Used to style webpages
JavaScript
Used to add functionality

8. Common Separators Used in Lists

HTML provides tags to separate content:

  • <hr> → Horizontal line separator
  • <br> → Line break
  • Nested list → Logical separator
  • Headings → Section separator
Example:

Line one
Line two

9. Avoiding Ambiguity (Which list to use?)

  • Ordered list → Use when order matters (steps, ranking)
  • Unordered list → Use when order doesn't matter
  • Description list → Use for definitions
