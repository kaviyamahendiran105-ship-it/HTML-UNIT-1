HTML Lists Tutorial

1. Ordered List

Definition: Ordered list shows items in numbered order.

Command Used:

<ol>
<li>Item</li>
</ol>

Example:

  1. Wake up
  2. Brush
  3. Study
  4. Sleep

2. Ordered List Types

Command Used:

<ol type="A">

Examples:

Capital Letters:

  1. HTML
  2. CSS
  3. JS

Small Letters:

  1. One
  2. Two

Roman Numbers:

  1. First
  2. Second

3. Unordered List

Definition: Unordered list shows items with bullets.

Command Used:

<ul>
<li>Item</li>
</ul>

Example:

  • Apple
  • Mango
  • Banana

4. Bullet Styles

Command Used:

<ul type="square">

Examples:

Disc:

  • Item
  • Item

Circle:

  • Item
  • Item

Square:

  • Item
  • Item

No Bullet:

  • Home
  • About

5. Nested List

Definition: List inside another list.

Command Used:

<ul>
   <li>Item
       <ul>
           <li>Sub Item</li>
       </ul>
   </li>
</ul>

Example:

  • Fruits
    • Apple
    • Mango
  • Vegetables
    • Carrot
    • Potato

6. Mixed List (Ordered + Unordered)

Example:

  1. Frontend
    • HTML
    • CSS
  2. Backend
    • Python
    • Node
