BLOCK vs INLINE ELEMENTS GUIDE

1. Block Elements

Definition: Block elements start on a new line and take full width.

Command Example:

<div>Block</div>

Example Output:

Block 1
Block 2

2. Inline Elements

Definition: Inline elements stay in same line.

Command Example:

<span>Inline</span>

Example Output:

Inline 1 Inline 2
